To Run (Visual Studio):

1. Open "HybridMobileProj" solution
2. Set "HybridMobileProj.Client" as startup project
3. Select IIS Express runtime
4. Takes 1-2 minutes to load up for first time
5. Get Mobile View
	5.1 - Open inspect tool and select mobile view port. OR
	5.2 - Reduce size of screen manually by dragging window
